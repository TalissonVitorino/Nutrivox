App Nutrivox: sistema de nutrição com app do paciente, ambiente do nutricionista, backend e módulo de IA.
- Nutricionista cria planos alimentares manuais, acompanha evolução, registra prontuário
- Paciente visualiza plano, registra consumo, consulta IA assistiva
- IA restrita ao domínio nutricional, nunca prescreve sozinha
- Stack: Kotlin Multiplatform + Jetpack Compose
- Fase atual: revisão de requisitos e design de produto (pré-implementação)

**Why:** Resolver problemas de apps de nutrição existentes (baixa clareza de plano, IA descontrolada, UX fraca)
**How to apply:** Toda decisão deve preservar autoridade do nutricionista, priorizar clareza do plano alimentar e manter IA como assistente
