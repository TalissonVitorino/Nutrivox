name: Design-First Process
description: Fluxo de desenvolvimento definido: revisão requisitos > design produto > UI externa > Figma > implementação
type: project
---

Processo definido pelo usuário:
1. Revisar e refinar requisitos da especificação (fase atual)
2. Transformar em proposta de design de produto
3. Levar proposta para ferramenta de IA gerar UI
4. Conectar ao Figma para consumir UI gerada
5. Implementar o aplicativo

**Why:** Garantir base sólida antes de implementar, evitar retrabalho
**How to apply:** Não produzir código até completar fases 1-4. Foco atual é análise crítica e design.
